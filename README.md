# Overview
This repo stores a linkwarden extension that I vibecoded with the goal of restricting its permissions so it wouldn't need the full network and browser access requested by the ones I saw on the market. It has the minimal permissions needed to get the current URL and communicate with linkwarden, and will request access to the linkwarden instance instead of requiring full network access.

This is an example of how AI can be used to empower individuals to accomplish tasks they were previously reliant on third parties for. I don't know a lot about browser extensions, but I was able to get it up and running in an hour or so (and learned a little along the way). It's definitely not something I'd call "high quality", but it gets the job done.
